You will see an emulator on the left half of the screen. You have to perform a clean up work, where if there is any unnecessary screens open then we close it. You have to act upon the following cases:

1 - If you see a keyboard open where there are characters a to z, on the mobile screen then on the bottom left corner of the mobile screen, you will see a down arrow key(coordinate at (125, 720)), press it. it will close the keyboard.
    Note that, an input field is not a keyboard.
2 - What happens sometimes is if you click on an input field, then there might be some pop ups that can show up for example, a drop down menu. To close such pop ups, consider pressing the back button. You can find the back button on the right verticle panel of the android studio. It looks like a triangle.
